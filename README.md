# 🐵 gokuu-new

## 目的

　本プロジェクトは、**憲法第９条に加憲するための理論と条文の構成**について考察し、HTMLを用いた
静的サイトとして公開します。　

　使用エディタ：TeraPad 
 
　構成：多層的なHTML構成で条文と理論を整理します。

## 加憲構成の概要

　**理論的根拠**   
  芦田修正を、二律背反構造の論理的解明で、その意図を明確にし、改憲案の策定を考察します。

　**必要性**   
  前文の理想と第９条の現実の乖離及び第９条が内在する問題を検証し、改憲の必要性を提示します。
  
  **構成の目的** 
1. 第３項は、第１・第２項の理論的解釈により引き継ぐことの正当性を立証し、国家防衛の根幹となる条文として構成します。
2. 第４項は、貿易立国の国策に伴い、国際平和の維持に貢献し、国際社会での信頼性の向上を目的として構成します。
3. 第５項は、解釈改憲で設けた防衛を所管する機関である、防衛省・自衛隊の正当性を確立することを目的として構成します。

  **各項の理論的考察**
1. 第３項は、第１・第２項の戦争の放棄、戦力及び交戦権の否認を全く正反対の国家防衛権行使を可能とするための
　加憲案を策定する立場から、第１・第２項の二律背反構造の概念であるを国家防衛権を本項で規定します。

２. 第４項は、国際社会の一員として果たすべき責務を憲法上に位置づけるもので、国際平和の維持に貢献し平和主義を
内向きの理念にとどめず、国際的実践へと昇華させる意義を有します。

３. 第５項は、文民統制の確保と組織運営の安定性を目的として、防衛機関を規定し戦前の統帥権独立の反省を踏まえた
　慎重な制度設計を志向するものであります。
 
## 考察・ディスカッション

2. 条文構成の改善提案
